Id in html

Id giống như class, cũng để chỉ định tên id cho 1 thẻ, id là duy nhất, tên id không được trùng nhau,id chỉ gồm 1 tên. ví dụ: id="box" là đúng còn id="box box2" là sai.

id được dùng để chọn 1 phần tử để style trong css hoặc để thao tác trong Js.
